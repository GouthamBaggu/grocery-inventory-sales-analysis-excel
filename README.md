# Grocery Inventory & Sales Analysis (Excel Project)

## Project Overview
This project analyzes grocery inventory data using Microsoft Excel to evaluate inventory risk, financial exposure, and profitability. The analysis includes exploratory data analysis (EDA), inventory risk monitoring, revenue analysis, and profitability metrics.

## Dataset
The dataset contains information about products, suppliers, stock levels, pricing, sales volume, and expiration dates.

## Key Analysis Performed
- Data Cleaning & Preparation
- Inventory Age and Expiry Analysis
- Reorder Monitoring
- Inventory Value Calculation
- Financial Risk Analysis
- Revenue & Profitability Analysis
- Exploratory Data Analysis (EDA)

## KPIs Created
- Total Inventory Value
- Inventory Risk Exposure
- Inventory Risk Percentage
- Reorder Financial Impact
- Revenue Contribution
- Profit Margin Analysis

## Visualizations
- Product Distribution by Category
- Supplier Contribution
- Stock Quantity Distribution
- Unit Price Distribution
- Sales Volume Distribution
- Stock Quantity vs Sales Volume

## Tools Used
- Microsoft Excel
- Pivot Tables
- Conditional Formatting
- Charts & Visualization
- Financial Calculations
